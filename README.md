- An Ecommerce Shopping Site created using MEAN Stack (Angular as fronend , Express as Backend)

 - It gives the ability to the user to fetch pizza details
 - Select multiple pizzas , toppings and even order Custom Pizza


   --------------------
   To start clone the repo , the inside the root folder

   start the frontend
   - do cd frontend
   - npm install
   - ng serve / npm start

  start the backend
 - do cd backend
 - npm install
 - npm start

connect to database
- In the mongo compass add
- a pizza json containing pizzas , ingredients .json containing toppings
